    세출 세부사업 예산편성현황(추경포함) - 예산은 추경이 반영된 자료입니다. - 추가경정예산은 예산이 성립한 후에 생긴 부득이한 사유로 인하여 이미 성립된 예산에 변경을 가하는 예산을 말하며, 세출은 국가 또는 지방자치단체가 한 회계년도 동안에 모든 수요에 충당하기 위하여 지출하는 일체의 경비를 말한다. - 지출은 일반적으로 세출예산 및 기금운용계획의 집행에 따라 국고에서 현금 등이 지급되는 것을 의미한다.

    세입/수입 예산편성현황(총액) 세입은 국가 또는 지방자치단체의 1회계 연도에 있어서의 국가 또는 지방자치단체의 모든 수요를 충족시키기 위한 지출의 재원(財源)이 되는 일체의 현금적 수입(收入)으로 '세출'(歲出)과 상반되는 개념이다. 세입은 조세에 의한 수입이 주류를 이루고 있으며, 그밖에 재산의 처분 또는 새로운 채무의 부담에 의하여 생기는 수입[공채 발행 등] 및 회계 간의 전입(轉入)과 국고 내의 이체(移替)에 의한 것도 포함한다. 그러나 우편예금의 예입이나 공탁금의 수입과 같이 지출의 재원이 되지 않는 것은 제외된다. 수입은 조세 등 국고금이 세입으로 납입되거나 기금에 납입되는 것을 의미한다.(국고금관리법, 제2조 2호)
    		연간기획재정부 예산실
    세입/수입 예산편성현황(총수입) 단년도, 세입/수입 예산편성 현황을 총수입 기준, 세입순계 기준으로 제공합니다. 1) 총수입 = 일반수입 - 금융성외평기금수입 2) 순계 - 세입순계 : 일반수입 + 보전수입 - 예산순계 : 예산수입 - (회계간거래 + 회계내계정간거래) - 기금순계 : 기금수입 - (회계간거래 + 회계내계정간거래)

    3. 3서비스 유형 선택 영역 : 서비스 유형별로 선택하여 조회하실 수 있습니다. 원하는 서비스 유형을 선택하여 클릭하면 우측 하단 목록 영역에 선택된 서비스 유형의 데이터 목록이 나타납니다.
    4. 4데이터 검색 영역 : 데이터셋을 검색 키워드를 통해 검색하여 조회하실 수 있습니다.
    5. 5분류 조회 설정 영역 : 분류별, 제목별, 생산기관별, 자료출처별로 사용자가 원하는 분류 형태를 선택하여 조회할 수 있습니다. 분류별은 분류 카테고리 순서로 나타나며, 나머지는 가나다순으로 조회됩니다. 하단 키워드 검색을 통해 원하는 분류 유형에 속하는 키워드 검색 결과를 조회할 수 있습니다.
    6. 6엑셀 다운로드 : 상세 재정통계로 제공되는 데이터 목록을 엑셀 파일로 다운로드 할 수 있습니다.
    7. 7분류체계별 선택 영역 : 분류체계별에 의해서 펼침, 닫힘 기능을 통해 카테고리를 확인할 수 있습니다.
    8. 8분류체계 상세 영역 : 좌측 탐색기 영역의 분류명을 클릭하면 해당 분류에 속한 데이터를 조회할 수 있습니다.
    TIP
    • 탭페이지는 총 5개까지 생성하실 수 있어 데이터 비교 분석이 용이하며, 가장 나중에 클릭한 데이터부터 앞에 위치하게 됩니다.
    • 좌측 탐색기 영역 상단의 펼침/닫기 글자 앞 ‘+’/’-’ 버튼을 클릭하면 모든 분류체계를 펼쳐 보이거나 닫을 수 있습니다.
    • 좌측 탐색기 영역앞 체크 박스를 클릭하면 해당 분류체계에 속하는 데이터를 우측 목록에서 확인할 수 있습니다.
    • 우측 데이터 목록 영역에서 복수의 서비스 유형을 가진 데이터의 경우 서비스 유형 정보에서 원하는 서비스 유형을 클릭하면 해당 서비스의 상세 페이지로 바로 이동할 수 있습니다.
    1. 1용어설명 : 버튼을 클릭하면 해당 데이터와 연관된 용어에 대한 설명 정보가 레이어 팝업으로 실행됩니다. (서비스 공통)
    2. 2메타정보 영역 : 해당 데이터에 대한 메타 상세 정보를 볼 수 있습니다. 버튼 내 우측 ▲▼ 표시를 클릭하여 화면 내에 상세정보를 보이거나 닫을 수 있습니다. (서비스 공통)
    3. 3서비스 영역 : 데이터의 특성에 따라 Sheet, Chart, OpenAPI, File, Link 서비스를 보실 수 있습니다. (서비스 공통)
    4. 4검색 영역 : 데이터를 조회조건에 따라 선택하여 보실 수 있습니다. 초기에는 해당 데이터의 검색 조건이 하나만 보여지며, 검색영역 하단 중앙의 ▲▼ 표시를 클릭하여 하단에 전체 검색조건이 보이거나 닫을 수 있습니다.
    5. 5다운로드 : 화면에 보여지는 데이터의 결과값을 다양한 파일 포맷으로 다운로드 할 수 있습니다.
    6. 6주석 설명 : 해당 데이터 정보에 대한 주석 설명을 볼 수 있습니다. (서비스 공통)
    7. 7관련 재정정보 : 해당 데이터와 연관된 데이터를 볼 수 있으며, 클릭 시 하단 관련 재정정보 리스트로 화면이 이동합니다. (서비스 공통)
    TIP
    • Sheet 영역 첫 줄 컬럼 헤더 영역에 컬럼명을 클릭하면 해당 컬럼명 기준으로 오름차순, 내림차순으로 데이터를 정렬할 수 있습니다.
    • 항목에 해당하는 결과값이 열 간격으로 제대로 보이지 않을 경우 컬럼명 옆 우측 라인에 마우스를 대고 좌우로 조정하면 열 간격을 조정할 수 있습니다.
    • 컬럼 순서를 변경하고 싶으면 컬럼명을 드래그하여 원하는 위치로 이동하여 볼 수 있습니다.
    1. 1차트설정 영역 : 차트를 사용자가 원하는 데이터를 기준으로 보려면 시리즈명에 원하는 항목을 선택하면 되며, 꺽은선 막대 등 총 7가지 다양한 유형으로 조회할 수 있습니다.
    2. 2범례 영역 : 사용자가 선택한 데이터 항목이 그래프 영역 하단에 범례로 표시되며, 특정 항목을 클릭하면 그래프 영역에서 값이 사라지거나 다시 볼 수 있습니다.
    TIP
    • 그래프는 사용자가 별도로 설정하지 않으면 모든 데이터 항목이 표시되므로 그래프를 통해 시각적으로 추이 등을 분석하기 어려울 수 있습니다. 따라서, 그래프 영역 하단 범례에서 원하는 데이터 항목을 선택하여 보시는게 좋습니다.
    • 그래프상에서 원하는 항목의 값을 보려면 마우스를 해당 영역으로 가져가면 됩니다.
    1. 1명세서 다운로드 : 화면상의 OpenAPI 개발명세 내용을 엑셀 파일로 다운로드 할 수 있습니다.
    2. 2인증키 신청 : Open API서비스를 이용하기 위해서는 먼저 인증키를 발급 받으셔야 합니다. 인증키는 회원에게만 제공되므로 반드시 회원가입과 로그인이 필요하며, 인증키 발급 후 별도의 승인절차 없이 바로 서비스를 이용하실 수 있습니다.
    3. 3개발명세 영역 : Open API 서비스를 활용하기 위한 요청주소, 제한횟수, 인자 정보 등을 제공합니다.
    4. 4샘플테스트 영역 : 실제 API서비스를 통해 제공받는 결과값을 테스트 등을 통해 확인할 수 있으며, 에러 또는 정상 완료 시 경우별 메시지 코드를 확인할 수 있습니다.
    TIP
    • API서비스는 사용량이 많을 경우 시스템에 부하가 발생할 수 있습니다. 따라서, 시스템 부하 등이 발생할 경우 호출 횟수 등을 제한할 수 있습니다.
